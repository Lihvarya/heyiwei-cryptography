# 何意味生成器

## 简介

“何意味生成器”是一个基于Web的趣味工具，它利用一组独特的中文词组（例如“何意味”、“河移位”等）对文本进行加密和解密。这个工具将输入的文本字符转换为一个特殊的“词组编码”，从而生成一串看似无意义但又带有特定模式的中文词组序列。

## 功能特性

*   **文本加密:** 将任何输入的文本内容转换为由预定义中文词组组成的密文。
*   **文本解密:** 将生成的密文还原回原始的文本内容。
*   **实时处理:** 提供快速的加密和解密操作（带少量延迟模拟处理）。
*   **剪贴板操作:** 支持从剪贴板粘贴输入，以及将处理结果复制到剪贴板。
*   **清空功能:** 方便用户清空输入框内容。

## 工作原理

本工具的核心在于一个自定义的“词组编码”系统：
1.  **词组定义:** 工具内部预设了一系列中文词组 (`PHRASES`)，这些词组构成了一个基数（`BASE`）。
2.  **字符转换:** 每个文本字符的Unicode编码被转换为一个基于`BASE`的数字表示。
3.  **词组映射:** 这个基于`BASE`的数字序列再被映射到对应的中文词组，从而生成密文。
4.  **解密过程:** 解密时，工具会识别密文中的词组，将其还原为基于`BASE`的数字，最终重构出原始字符的Unicode编码，完成解密。

## 如何使用

1.  **打开文件:** 下载 `何意味.html` 文件，并在您的Web浏览器中打开它（例如 Chrome, Firefox, Edge）。
2.  **输入文本:** 在“输入文本”区域的文本框中输入您想要加密或解密的文字。
3.  **选择操作:**
    *   点击 **“加密”** 按钮，将您的文本转换为词组密文。
    *   点击 **“解密”** 按钮，将输入的词组密文还原为原始文本。
4.  **查看结果:** 处理结果将显示在下方的“处理结果”区域。
5.  **辅助功能:**
    *   使用“粘贴”按钮从剪贴板快速输入。
    *   使用“清空”按钮清除输入框内容。
    *   使用“复制”按钮将处理结果复制到剪贴板。

## 本地部署

由于这是一个纯HTML、CSS和JavaScript项目，您无需任何服务器环境即可运行。
1.  克隆或下载此仓库。
2.  在任意浏览器中打开 `何意味.html` 文件即可。
